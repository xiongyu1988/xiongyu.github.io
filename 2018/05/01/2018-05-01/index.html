不同频段的声固耦合问题分析方法及波基法(Wave Based Method)简介

  2. 2. 高频段 High-Frequency Band

  4. 4. 波基法 Wave Based Method 简介

对于经典的振动声学耦合问题，根据针对不同的目标频段内表现出的响应特征有较大差异，因此，一般情况下在不同频段内研究的振动声学耦合问题需要用到不同的求解方法。一般目标频段可分为三段：低频段，中频段和高频段。

低频段 Low-Frequency Band

低频率可定义为所研究模型的特征尺度远小于该频率下系统动力学响应的特征波长，此特征波长可为系统中声波波长或者结构弹性波波长，或者与之在同一量级上。在这个频率段，系统的模态具有明显的界限特征，“模态交替 (Modal Overlap)” 现象很弱。在这种条件下，使用传统的确定性方法能够精确的求解系统的振动声学响应。因此，传统基于单元的数值计算方法在此频段内得以广泛应用，最为常见的包括有限元方法 (Finite Element Method)和边界元方法(Boundary Element Method)两种。前置可以用于计算整个系统的振动声学响应，而后者则多用以计算结构振动所引起的辐射声。

高频段 High-Frequency Band

当所研究的模型特征尺寸远大于该频率下系统动力学响应的特征波长时，整个系统的模态分布会变得极其密集，且不同模态间的互相影响变得明显，这样的频率范围一般即被定义为高频段。此时，系统的响应对于微小扰动的敏感性会急剧升高，在这样的情况下，工程上转而使用统计学的方法对系统的响应特性进行研究，因为以平均状态衡量系统的振动声学特性相较于单一确定状态而言更具实用性。由此，统计能量法(Statistical Energy Analysis)被广泛的应用于高频区间相关的系统动力学分析中。

中频段 Mid-Frequency Band

根据上述对低频段和高频段的定义，位于二者之间的，通常被称为中频区间。从大量的时间经验来看，此区间包含的频率范围与正常人类的听觉范围(20Hz-20kHz)大部分重合，故研究此频率范围内的振动声学问题尤其重要。为了保证求解精度，对于纯声学问题，常用基于单位的求解方法在使用上一般要求在一个物理波长尺度内至少含有8-10个单位，这一原则在主要以空气为传声介质的声学问题中似乎并无不妥，但是一旦涉及结构声学耦合问题，就不得不讲结构中的弹性波进行一并考虑。在一般情况下，结构中的弹性波的波长远小于同频率下空气中声波的波长，由于存在两者之间的直接耦合，此时理论中上要求系统单元尺度的选取需要以结构中弹性波波长为标准。如此，随着目标频率的升高，系统单元数量会急剧增加，极大降低计算机求解的计算效率。为了改善在中频段内的计算效率和准确度，目前最为常用的方法是扩大基于单位的数值算法的使用频率范围，例如改进求解器，区域分解(Domain Decomposition)和多极子方法(Multipole Method)等，这些方法基于分块近似对系统进行解构和求解的。与这些方法不同的是，还有另外一类方法基于系统控制偏微分方程的解析解对系统进行解析构建和求解，例如谱单元法(Spectral Element Method)和传统的Trefftz方法。

波基法(Wave Based Method)其本质上属于一种间接的Trefftz方法，它将各阶波函数贡献系数代替系统真是物理自由度进行求解，在具体应用方式上，波基法与传统基于单元的数值方法有一个共同点，即都是使用预先已知的的型函数(基函数)去近似整个系统的物理量，通过边界条件的约束构成封闭方程进行求解。不同的是，有限元类方法大多使用人为设计的多项式作为型函数，而波基法则使用控制方程的解析解作为基函数。对于基于单元的计算方法而言，需要将整个计算域划分为数量众多的小单元，理论上讲，其计算效率与划分的单元数量直接相关。波基法则是讲整个计算域划分为几个子域进行计算，这一点有些类似于区域分解法，没一个子域中的目标物理量均可以表述为一系列波函数(wave function)的加权求和，其中没一个波函数均为域内动力学控制方程的解析精确解。同时，波基法通过加权残差函数控制边界上的数值精度及相邻域边界上的连续性。通常情况下，有限单元法可以通过增加单元数量提高计算精度，而波基法则通过增加波函数的数量提高计算精度。从形式上来看，波基法构建的系统自由度要远少于传统的有限元方法与边界元方法，单就这一点而言，意味着前者将具有更高的计算效率。另一方面，波基法使用目标计算域控制方程的解析解作为基函数，由于其基函数系的完备性，一次理论上对全频段皆具有较高的求解精度。

波基法 Wave Based Method 简介

1998年基于传统的间接Trefftz方法，比利时鲁汶大学的Desmet在其博士论文中首次提出了波基法(Wave Base Method)这一概念，并给出了这一方法用于求解单腔结构声学耦合问题的基本理论及思路。波基法的基本思路是将计算域中的基本物理量展开成一系列波函数叠加的形式，其中没一个波函数均为系统控制方程的解析精确解。正是由于使用了控制方程解析精确解的方式，故针对高阶物理变量的计算不会存在精度损失。此外，由于波基法并不基于对系统进行离散单元划分，在计算声固耦合系统动力学响应问题时的计算效率仅仅取决于该目标频率所对应需要的波函数隔声，从系统自由度的角度出发，波基法系统的自由度数目将远少于传统基于离散单元的计算方法，如有限元。一方面，考虑到波基法的解析优势，与有限元方法一样，波基法理论上可以被运用于求解任意几何形状系统的振动声学特性问题。截止目前为止，波基法已被用于求解内声场问题，声辐射与声散射，弹性薄板振动及简单弹性板腔类结构声学耦合等问题的分析求解中。不过另一方面，由于基于波基法的耦合系统动力学方程的简历需要对目标几何进行相应的边界积分，因此该方法尚未用于处理拥有较为复杂几何拓扑结构的声固耦合系统中。
