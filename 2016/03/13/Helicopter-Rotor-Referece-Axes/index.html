直升机旋翼参考面 (Reference Axes)

有许多物理平面可以用来表述运动方程。

中心面 (Hub Plane - HP)

这个平面与旋翼轴垂直。在此平面中，在向前飞行状态中的循环挥舞运动 (Cyclic Flap Motion) 和循环变矩运动 (Cyclic Feather Motion) 可以被很好观测到。这个平面可以很好的用来桨叶运动分析 (Blade Dynamics Analysis)。

非变矩面 (No Feather Plane - NFP)

这是一个无法观察到循环俯仰角 (Cyclic Pitch) 变化的平面，所以，$ \theta_{1c} $ 和 $ \theta_{1s} $ 都等于 $ 0 $。这个平面通常用来直升机性能和稳定性分析。将会有挥舞角 (Flap Angle) 的循环变化。在Gessow 和 Myer的书中，此平面被用于直升机性能的研究。

桨尖面 (Tip Path Plane - TPP)

这个平面由桨尖 (Blade Tips) 表述而成，所以将没有俯仰角 (Pitch) 循环变化，所以，$ \beta_{1c} $ 和 $ \beta_{1s} $ 都等于 $0$。此平面通常用于气流通过旋翼时的学习。

控制面 (Control Plane - CP)

此平面也可称为可控总矩平面 (Commended Cyclic Pitch Plane)，并且通常可表示为桨毂 (Swash Plate) 平面。

在不同平面下的分析可以在不同的平面之间进行转换。

挥舞 (Flapping) 和周期变矩 (Feathering) 相等性

$\beta_{1c} + \theta_{1s} = Constant$
$\beta_{1c} + \theta_{1s} = (\beta_{1c})_{NFP}$
$\beta_{1c} + \theta_{1s} = (\theta_{1s})_{TPP}$

$\beta_{1s} - \theta_{1c} = Constant$
$\beta_{1s} - \theta_{1c} = (\beta_{1s})_{NFP}$
$\beta_{1s} - \theta_{1c} = - (\theta_{1c})_{TPP}$

使用这些转换关系可以用来转换分析式从一个参考系到另一个参考系、

$\lambda = \lambda_{NFP} + \mu \theta_{1s} = \lambda_{TPP} - \mu \beta_{1c}$
$\alpha = \alpha_{NFP} + \theta_{1s} = \alpha_{TPP} - \beta_{1c}$

$H = H_{NFP} + \theta_{1s} T = H_{TPP} - \beta_{1c} T$
$Y = Y_{NFP} - \theta_{1c} T = Y_{TPP} - \beta_{1s} T$

$\lambda = \mu \alpha + \lambda_{i} = \mu (\alpha_{NFP} + \theta_{1s}) + \lambda_{i}$
$\lambda = \mu \alpha + \lambda_{i} = \mu (\alpha_{TPP} - \beta_{1c}) + \lambda_{i}$
