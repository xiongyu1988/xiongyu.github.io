New Guangzhou GND Sector File

2015-05-21 Flight Simulations Views

I was working on it for these two days.

Since last Thursday, Bridge (A name of Flight Simulation Developer Team) Guangzhou International Airport Scenery Add-on for Flight Simulation X and Prepar3D released. So, it is no doubt, as an original developer of Guangzhou GND sector file for both VRC and Euroscope version, I have to make a update for my previous work, Guangzhou Ground Sector file.

Here below is my improved-version for ZGGG GND sector. Some of bugs and errors have been fixed and also some improvements have been done. Coordinates of this sector have been tested with this new-released Guangzhou scenery and they are matched perfectly. In brief, it is much better than previous one.
