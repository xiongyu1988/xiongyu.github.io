相位，相速度，群速度

2016-10-28 Acoustic Views

  1. 1. 相位（Phase）
  2. 2. Phase Velocity 相速度
  3. 3. Group Velocity 群速度
  4. 4. Dispersive Wave 频散波

相位（Phase）

相位是描述信号波形变化的度量，通常以度作为单位，也称为相角。当信号波形以周期的方式变化，波形循环一周即为360度。

如简谐运动中：
$$x(t)=A cos(2\pi f t + \phi) $$
$$y(t)=A sin(2\pi f t + \phi) = A cos(2\pi f t + \phi - \frac{\pi}{2})$$

A 为振幅；$\omega = 2 \pi f$ 为角频率；t 为时间；$\phi$ 为初相角或简谐相角；$\omega t + \phi$ 为在时刻t振动的相或相位。

Phase Velocity 相速度

波的相速度，相位速度，或者简称相速（Phase Velocity）是指波的相位（Phase）在空间中传播的速度。之前介绍得知相位（Phase）是指波形变化的度量，对于一个正弦波或者一个余弦波一个周期为360度，相位即指这个波上的一个质点当位置，此位置以360度中其中的一个度表示，比如当前质点位置为90度位置，所以可得知这个质点在这个正弦波第一个波峰的位置。

可以挑选波的任意特定相位来观察，例如其中一个波峰，这个波峰会以相速度前行。相速度可有波的频率$f$乘以波长$\lambda$，或者角频率$\omega$除以波数$k$来计算得出。
$$v_{p} = f \lambda = \frac{\omega}{k}$$

Group Velocity 群速度

注意波的相速度和群速度比一定相等！相速度（Phase Velocity）是指波中一个相位移动的速度；群速度（Group Velocity）是指波中振幅变化的速度。

群速度和相速度只有在混合波（非单频波）在频散介质中传播时才有差别。

Dispersive Wave 频散波

频散波，是指若相速不仅依赖于介质的物理性质，还依赖于波数，这种波称为频散波，并称介质为频散介质（Dispersive Medium）。

若相速不仅依赖于介质的物理性质，还依赖于波数，这种波称为频散波，并称介质为频散介质。若相速仅依赖于介质的物理性质，不依赖于波数，这种波称为非频散介质，并称介质为非频散介质。


此图为深水表面重力波的传播模式。红点位相速度运动，绿点为群速度运动。这个例子中，红点冲左向右运动的过程中两次跨过绿点，相速度约为群速度的两倍。新的波看起来像是从一个波群的末尾开始生产，振幅逐渐增大直至到达波群中间，然后再消失于波的前端。对于水表面的重力波，大多数情况下群速度远小于相速度。
